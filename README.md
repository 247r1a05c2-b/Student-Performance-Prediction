# Student Performance Prediction using Multiple Linear Regression

## 📌 Project Overview

This project was developed as part of the **KodBud AIML Internship – Task 3**. The goal of the project is to predict a student's final grade based on factors such as study hours, attendance, previous scores, and assignment performance using **Multiple Linear Regression**.

By analyzing these factors, the model can estimate academic performance and identify the variables that have the greatest impact on student success.

---

## 🎯 Objectives

- Analyze student performance data
- Perform data preprocessing and exploration
- Train a Multiple Linear Regression model
- Predict final student grades
- Evaluate model performance
- Identify the most influential factors affecting academic performance

---

## 📊 Dataset Features

The dataset includes the following attributes:

- Study Hours
- Attendance Percentage
- Previous Scores
- Assignment Scores
- Final Grade (Target Variable)
